# carl-the-llama

> A GitHub App built with [Probot](https://github.com/probot/probot) that is meant for enforcing rules on his own repository.

## Setup

```sh
# Install dependencies
npm install

# Run the bot
npm start
```

## Features

Things Carl does:
- Leaves comments on new issues
